## ✨ What It Does

Instead of navigating complex government portals, users simply **type a question** and the chatbot:
- 🔍 Searches a vector database of government schemes using **semantic similarity**
- 🧠 Passes the most relevant schemes to **Google Gemini AI** as context
- 💬 Returns a **factual, grounded answer** — no hallucinations, no guessing
